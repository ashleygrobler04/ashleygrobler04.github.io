This guide aims to instruct the user how to use the stock management program.


Upon first launch.

When first launching the program, you will be greeted with some text, refering you to this manual if you are stuck at any point.

Adding an item.

Adding an item is easy. Follow these steps:

  1. In the menu, navigate to the "Manage stock" menu, expand it and click on "stock manager."
  2. A new window should pop up. Maximize it just incase you don't see all the items.
  3. Enter a name for your item in the dedicated textField.
  4. In the next textField, enter a price for your item. This can either be with the format (10.15) or 10
  5. Using the next form control, choose the date the item is added or leave it at it's default value which should be the current date.
  6. If you wish to, you can add an image to describe the item, click on the "Add image" button and browse for it. Note: Please don't click on the button and cansle the operation as it may freeze the program then you would have to re-add the item. If you don't want to add an image, skip this step.
  7. Lastly, click on the "Add" button to add your item as a record to the database and watch the table update.

Removing an item

Removing an item is easy too, All you have to do is move your mouse into the table to the row containing your item, then move to the "Remove Selected Item" button and click it.
Upon clicking on the button, you should be asked if you want to delete the spessified item. Conferm by clicking "Yes" or cansle by clicking "no".

Incrementing and decrementing an item

This can be done by moving your mouce to the row containing your item and clicking on the increment/decrement button. This will update the amount (quantity) of the item.

Searching for an item.

Searching for an item is something you might use often.
All you need to do is go to the "What are you searching for" field and enter the name of the item you are looking for and press the "Search" button.

Export to excel

You can export the table containing all your items to a spreadsheet by clicking the "export to excell" button. In the dialog that pops up, enter a name for your file and choose a location to save it to.

Changing the background color

To change the background color of the program, locate the control "change backGround color", and select any available option.
